影像共享平台

  • 安利
  • 2019-09-24

目标

搭建一个支持图片分享以及部分用户图片上传的网站。

起因

近期一位稍微比我富裕那么一点的巨佬Algha_Porthos，由于要满足Image Sharing Platform（影像共享平台）的需求，现将Algha_Porthos，Oblivion，joefang三人组成技术支持组，进行网站的设计工程。

介绍

目的：

希望建立平台以共享已经公开却难以找到获得途径的照片，并且给希望公开照片的人一个平台。

平台基本性质：

本平台将会是一个基于互联网的影像共享平台，将实现用户不同权限、实时可访问、下载、持权上传影像（在保证安全后，可能拓展到所有文件）。目前计划在7年内保持正常运营。服务器是阿里云学生机，我们将以云盘或者图床为基础进行权限设置。

服务器的性能：

镜像系统：Windows 2012 R2
流量：1000GB
CPU：一核
内存：2GB
系统盘：40GB

用户权限设置：

目前计划分为3~4个等级如下：

等级 访问方式 上传权限 下载权限 用户授权权限 紧急关站
Owner 账号+密码 有 有 有 有
Administrator 账号+密码 有 有 对等级低者有 无
User 账号+密码 联系有权限者帮忙上传 有 无 无
Tourist（可无） 公共账号 无 限速下载部分文件 无 无

资金来源

初期测试资金全部由Algha_Porthos承担，后期维护服务器的资金以Algha_Porthos为主，博主和joefang也会承担一部分资金，当然欢迎各位用户对本项目进行赞助，我们也会设置关于赞助的致谢名单，也可以应赞助者的要求在网站上添加友链。

项目调试日志

理论阶段

日期：2019-9-16到2019-9-22
确定需求，选择需要购买的服务器以及讨论如何实现需求。

测试阶段

No1.极简网盘

日期：2019-9-23
支持文件上传的后缀限制。
支持用户权限设置，分为user，admin，superadmin三种权限。
支持多语言，包括中文，英文，日文。
暂时因为邮箱验证原因不支持用户注册，只能后台注册用户。

No2.云存储个人网盘

日期：2019-9-25
支持所有文件上传以及分享。
支持用户任意注册。
不支持权限设置。

No3.kiftd云盘

日期：2019-9-25
支持权限设置。
支持所有文件上传。
需要java环境，无法在虚拟主机上运行，可以在本地运行并且访问，未在服务器环境下测试。

No4.Cloudreve

日期：2019-9-25
测试用虚拟主机无法安装宝塔。
预计国庆期间在服务器上测试。

No5.蓝眼云盘

日期：2019-9-25
需要操作系统支持。
预计国庆期间在服务器上测试。

No6.nextcloud

日期：2019-9-25
需要宝塔支持。
预计国庆期间在服务器上测试。

No.7Amoli

日期：2019-9-25
仅支持下载操作，后台上传。
无用户系统支持。

实际搭建阶段

未开始，预计在国庆期间开始。

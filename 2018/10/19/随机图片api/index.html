什么是随机图片api

随机图片api是什么呢？通俗的讲就是当你输入一个网址浏览器时会返回给你不同的图片。
其实原理很简单，把你要随机的图片放在一起，然后写一个php，当php被访问时，就随机指向这些图片中的一个返回给浏览器。

我的随机图片api

教完你们之后就要推广一下我的随机图片啦。
由于本蒟蒻是一个从来不氪金的乖孩子（其实是因为我穷），
我的api就接在qyy大佬的服务器上了。
所以我使用的域名也是他的，不要惊讶啊。
https://cf.zjqyy.top/img.php
由于我想把a随机图片运用到背景上面，所以我专门分类了一个背景图片的api：（许多图片的长宽比例不适合作为背景，所以背景随机图片有点少）
https://cf.zjqyy.top/background/img.php
图片来源是p站，收集图片耗费了我近三个小时（其实是小飞机不给力，才下载了一百多张图片，毕竟p站是要翻墙的）。
图片后续还会增加，可能会有点慢，毕竟爬虫是无法分辨图片质量如何的（其实是我太蒻了不会写爬虫）。
如果你有更好的高清图片（一定要高清哦），或者发现我的背景随机图片不适合做背景，请联系我，把网址发给我。

Hexo 指令浅谈

简写对照

n--new
p--publish
g--generate
s--server
d--deploy

安装

npm install hexo -g

更新

npm update hexo -g

初始化

hexo init

部署

hexo d -g//上传

本地编辑

hexo s
在http://localhost:4000/查看

插入

hexo n post "TITLE": 添加博客.
hexo n page "TITLE": 添加分栏.
